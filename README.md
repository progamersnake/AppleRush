# 🍎 AppleRush

Catch all the apples! A fun web-based arcade game where you control a basket to catch falling apples.

## Game Features

- 🎮 **Simple Controls**: Use arrow keys (LEFT/RIGHT) or A/D keys to move the basket
- 🍎 **Catch Apples**: Click on apples or catch them with the basket
- 📈 **Progressive Difficulty**: Each level gets harder with more apples and faster falling speed
- 🎯 **Scoring System**: Earn points for each apple caught (bonus points at higher levels)
- ❤️ **Lives System**: You get 3 lives. Lose a life when an apple falls without being caught
- 📊 **Score Tracking**: Monitor your score, lives, and current level

## How to Play

1. Click the **"Start Game"** button to begin
2. Use **LEFT** and **RIGHT** arrow keys to move the basket left and right
3. Catch falling apples by clicking on them or catching them with the basket
4. Complete each level by catching all apples that spawn
5. Progress through increasingly difficult levels
6. Game ends when you run out of lives

## Controls

- **Arrow Keys (← →)** or **A/D** - Move basket left/right
- **Mouse Click** - Click apples to catch them
- **Start Button** - Begin the game or restart

## Game Mechanics

- Each level spawns a fixed number of apples
- Apples fall from random positions at the top
- Missing an apple costs you 1 life
- Each level has more apples and they fall faster
- Score multiplier increases with level
- Catch all apples to advance to the next level

## Technical Stack

- **HTML5** - Game structure and markup
- **CSS3** - Styling and animations
